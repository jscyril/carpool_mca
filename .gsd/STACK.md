## Key Design Decisions

- **Passwordless auth** — No password hashing required; JWT only after OTP verification
- **Async-first backend** — All DB operations use SQLAlchemy async + asyncpg
- **Pluggable external services** — SMS, Email, OCR, Verification all switchable via `PROVIDER` env vars (console mode for local dev)
- **PostGIS for geospatial** — Ride start/end stored as `Geography(POINT, SRID=4326)` for accurate distance queries
- **No Alembic** — An `init_db.py` script exists instead; migration tooling not confirmed
